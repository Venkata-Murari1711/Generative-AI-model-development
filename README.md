### **This repository contains:**
- **Assignment 1:** Discriminative (Logistic Regression) and Generative model (GAN)
   - **Dataset:** MNIST dataset
- **Assignment 2:** Transformer Architecture (GPT-2, BERT, and T5 model)
   - **Dataset:** CNN Dailymail
- **Assignment 3:** Prompt Engineering Lab
   - **Model:** LLaMA 3.2 (1B) deployed locally via Ollama
   - **Task:** Summarization of a fixed paragraph
- **Assignment 4:** Comparative Training Methods (Pre-Training (unsupervised), Supervised Fine-Tuning (SFT) and Reinforcement Learning)
   - **Model:** Tiny GPT
   - **Dataset**: WikiText-2
